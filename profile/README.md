# Leck-Eier-AG

Welcome to **Leck-Eier-AG**, a meme company for fun projects, weird ideas, and unnecessary engineering done with maximum style.

## What We Do
We build joke projects that are still technically solid.
Think:
- parody tools
- absurd automations
- overengineered minisites
- experiments that started as "what if..."

## Why This Exists
Because not everything in software needs to be serious.
This space is for learning, laughing, and shipping creative nonsense that still runs.

## Project Principles
- Keep it fun.
- Keep it lightweight.
- Keep it shippable.
- Keep it respectful.

## Contributing
Got a ridiculous idea? Open an issue or submit a PR.

Suggested flow:
1. Propose the meme concept in an issue.
2. Keep the scope small and runnable.
3. Add a short README to your project folder.
4. Ship it.

## House Rules
- No secrets in commits.
- If it breaks, document it.

## Disclaimer
Leck-Eier-AG is a fictional meme company.
No business services, no legal promises, just fun builds.
